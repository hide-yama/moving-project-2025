# ファーストボックス

作成日: 2025年10月28日
プラン: **フルコースプラン**（荷解きまでアートが対応）

---

## 📦 ファーストボックスとは

引っ越し当日に自分で持ち運び、すぐにアクセスできるようにしておく荷物。
アートのスタッフに梱包してもらわず、自分で管理するもの。

---

## 🎯 フルコースプランの場合の考え方

### 前提
- **11月15日当日**: 荷解きまで完了する
- **夕方～夜**: 食器も日用品も棚や引き出しに配置されている
- **重点**: 貴重品管理 + 当日夜の緊急対応

### ファーストボックスが必要な理由

#### シチュエーション1: 作業中にスタッフに触れてほしくないもの
- **印鑑、通帳、マイナンバーカード、現金**
  - 当日の支払いに印鑑が必要
  - 紛失リスクを避けるため自分で管理
- **下着類、プライベートなもの**
  - スタッフに梱包してほしくないもの（事前に自分で段ボールに詰める）

#### シチュエーション2: 作業中すぐに必要になるもの
- **スマホ充電器**
  - 作業中も連絡が取れる状態を維持
  - スタッフとの連絡、写真撮影など
- **ペットボトルの水・お茶**
  - 作業中の水分補給（コップが使えない状態でも飲める）

#### シチュエーション3: 荷解き後の「どこに何があるか分からない」問題
- **洗面・入浴用品一式**
  - 歯ブラシ、シャンプー、タオルなど
  - 荷解きされていても、どこにしまわれたか分からない
  - 夜すぐにシャワーを浴びたい、歯を磨きたい
- **トイレットペーパー**
  - スタッフが帰った直後にトイレに行きたくなる可能性
  - 配置場所を自分で決めたい

#### シチュエーション4: 翌朝の生活立ち上げ
- **着替え（パジャマ、翌日の服）**
  - 荷解きされていても、すぐに見つからない可能性
  - 朝バタバタせずに準備したい
- **髭剃り**（男性）
  - 翌朝の身支度に必要
- **生理用品**（女性・タイミング次第）
  - 必要な場合にすぐ取り出せるように

#### シチュエーション5: 万が一のトラブル対応
- **常備薬（頭痛薬、胃腸薬など）**
  - 疲労で体調を崩す可能性
- **ハサミ/カッター、簡単な工具**
  - スタッフが帰った後に何か調整が必要になる場合
- **水・軽食**
  - 夜遅くまで作業、食器の位置確認が面倒な場合

## ✅ ファーストボックス推奨リスト

**コンセプト**: 「1泊2日の旅行セット」
荷解きは終わっていても、**どこに何がしまわれているか分からない**リスクに備える

### 【必須】貴重品・当日必要
- [ ] **印鑑** ← 当日の支払いに必要
- [ ] **現金・クレジットカード**
- [ ] **通帳、マイナンバーカード**
- [ ] **スマホ充電器**

### 【重要】1泊分の生活用品（旅行セット感覚）
#### 洗面・入浴
- [ ] **歯ブラシ・歯磨き粉**
- [ ] **洗顔料**
- [ ] **シャンプー・コンディショナー（旅行用サイズ）**
- [ ] **ボディソープ（旅行用サイズ）**
- [ ] **髭剃り**（男性）
- [ ] **生理用品**（女性・タイミング次第）
- [ ] **タオル（バスタオル1枚、フェイスタオル2枚）**
- [ ] **基礎化粧品**（必要な人）

#### 着替え
- [ ] **パジャマ**
- [ ] **翌日の着替え1日分（下着含む）**
- [ ] **靴下**

#### トイレ
- [ ] **トイレットペーパー（1ロール）**

### 【推奨】万が一の備え
- [ ] **常備薬（頭痛薬、胃腸薬など）**
- [ ] **ハサミまたはカッター**
- [ ] **ドライバー（プラス・マイナス）**
- [ ] **ペットボトルの水・お茶**
- [ ] **軽食（カロリーメイトなど）**

## 💡 ハーフコースとの違い

| 項目 | ハーフコース | フルコース |
|------|-------------|-----------|
| 荷解き状況 | 段ボールのまま | 荷解き完了 |
| 主な問題 | **段ボールを探す必要** | **どこに何があるか分からない** |
| 必要量 | 多め（1～2日分の生活必需品） | 1泊2日の旅行セット |
| 重点 | **生活の立ち上げ** | **貴重品管理＋すぐ使うもの** |
| イメージ | 避難袋 | 旅行バッグ |

## 📦 準備方法

- **容器**: 旅行バッグやキャリーケース（1泊2日の旅行をイメージ）
- **準備期限**: 11月8日まで
- **保管場所**: 当日まで自分で管理、引っ越し当日は車で運ぶ
- **ポイント**:
  - 洗面用品は旅行用サイズを購入すると便利
  - 100円ショップやドラッグストアで小分けボトルも活用
  - 普段の旅行準備と同じ感覚でOK
