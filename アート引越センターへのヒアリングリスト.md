# アート引越センターへのヒアリングリスト

作成日: 2025年10月26日
見積もり訪問時の質問事項

## 📦 プラン・料金関連

### プランの比較
- [ ] **フルコースプラン**の見積もり金額
  - 梱包作業（引越し前）
  - 搬出・搬入
  - 開梱作業（引越し後）
  - 段ボールの回収

- [ ] **ハーフプラン**の見積もり金額
  - 梱包作業（引越し前）
  - 搬出・搬入
  - 開梱作業なし（自分たちで行う）

- [ ] 両プランの料金差額と作業内容の違い

### 追加オプション・サービス
- [ ] 段ボールの提供数（無料/有料の範囲）
- [ ] 梱包資材（緩衝材、テープなど）の提供内容
- [ ] 不用品回収サービスの有無と料金

## 🧺 洗濯機関連

- [ ] **現行洗濯機の処分費用**（引き取ってもらう場合）
- [ ] **洗濯機を新居に運ぶ場合**
  - 搬入・設置サービスの有無
  - 設置作業の料金（標準設置の範囲）
  - 給水ホース・排水ホースの接続まで対応してくれるか
  - 水漏れテストまで実施してくれるか

## 🛠️ 引越し当日までの準備作業

### 解体が必要な家具
- [ ] ベッド（解体の要否）
- [ ] 本棚・収納棚（解体の要否）
- [ ] テーブル・デスク（解体の要否）
- [ ] その他大型家具の扱い
- [ ] **どの家具を事前に解体しておく必要があるか？**
- [ ] **解体はアート側でやってくれるのか、こちらでやるのか？**

### 壁・床の原状回復準備
- [ ] 壁に貼ってあるもの（フック、粘着テープ、ポスターなど）を剥がしておく必要があるか
- [ ] 床に敷いてあるマット類は外しておく必要があるか
- [ ] カーテンレールからカーテンを外しておく必要があるか

### 小物・散らばっているものの整理
- [ ] 引き出しの中身はどの程度まとめておく必要があるか
  - そのままでOK？
  - ある程度まとめる必要がある？
  - 全て段ボールに詰める必要がある？
- [ ] クローゼット内の衣類の扱い
  - ハンガーにかけたままでOK？
  - 畳んで段ボールに入れる必要がある？
- [ ] 本棚の本は抜いておく必要があるか
- [ ] 食器棚の食器は取り出しておく必要があるか

### 家電関連
- [ ] **冷蔵庫の中身**
  - 食品は全て空にする必要があるか
  - 調味料などもすべて出す必要があるか
  - 中身を入れたまま運搬できるか
  - 当日の食品管理方法（クーラーボックスを用意すべきか）
  - どのくらい前から食品を減らしていくべきか
- [ ] 冷蔵庫の水抜き・霜取りのタイミング（前日？当日朝？）
- [ ] 冷蔵庫の電源を切るタイミング
- [ ] 洗濯機の水抜きのタイミングと方法
- [ ] エアコンの取り外し・取り付けサービスの有無と料金
- [ ] テレビ・オーディオ機器の配線は外しておく必要があるか

### その他準備事項
- [ ] 梱包しなくて良いもの・してはいけないもの
  - 貴重品（現金、通帳、印鑑など）
  - 重要書類
  - その他
- [ ] ゴミ（燃えるゴミ・燃えないゴミ）は処分しておく必要があるか
- [ ] 植物（観葉植物など）の運搬可否

## 🏠 物件関連の確認

### 現居（新座市）
- [ ] 搬出経路の確認（玄関、廊下、階段、エレベーター）
- [ ] トラックの駐車スペースの確認
- [ ] エレベーターの有無とサイズ
- [ ] 搬出時の養生範囲

### 新居（小金井市前原町）
- [ ] 搬入経路の確認（玄関、廊下、階段、エレベーター）
- [ ] トラックの駐車スペースの確認
- [ ] エレベーターの有無とサイズ
- [ ] 搬入時の養生範囲
- [ ] **ピアノの搬入経路と作業可否**（ピアノ持ち込み予定のため）
- [ ] 大型家具（ベッド、冷蔵庫など）の搬入経路確認
