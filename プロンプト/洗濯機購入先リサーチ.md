# 洗濯機購入先リサーチ用プロンプト

作成日: 2025年10月26日

---

## AIへの指示プロンプト

```
パナソニックのドラム式省スペース洗濯機（価格帯: 約170,000円）の購入について、以下の点を調査してください。

【購入する製品】
- メーカー: パナソニック
- タイプ: ドラム式、省スペース（コンパクト）モデル
- 価格帯: 約170,000円

【状況】
- 引越し予定日: 2025年11月15日（土）
- 新住所: 東京都小金井市前原町一丁目6番の4
- 現在の洗濯機の撤去も必要

【調査してほしい内容】

1. **購入先の比較**
   - ヨドバシカメラ、ビックカメラ、ヤマダデンキ、ケーズデンキ、Amazon、楽天市場など
   - 各店舗の価格（ポイント還元含む実質価格）
   - 配送・設置サービスの有無と費用
   - 古い洗濯機の引き取りサービスの有無と費用

2. **購入タイミング**
   - 今すぐ購入すべきか、セール時期を待つべきか
   - 11月15日の引越しに間に合うように配送・設置できるか
   - 在庫状況

3. **撤去・設置サービス**
   - 古い洗濯機の撤去費用（各購入先での比較）
   - 新しい洗濯機の設置費用（標準設置工事の内容）
   - 設置場所への搬入経路の事前確認の必要性

4. **推奨される購入方法**
   - 最もコストパフォーマンスが良い購入先
   - トラブル回避のための注意点
   - 保証内容の比較

【求める回答形式】
- 購入先ごとの比較表
- 具体的な総額（本体価格 + 配送費 + 設置費 + 撤去費 - ポイント還元）
- 推奨される購入先とその理由
- 購入タイミングの提案

2025年10月26日時点での最新情報で調査してください。
```

## 使用メモ

- 使用するAI: o3 MCP / ChatGPT / その他のリサーチAI
- 目的: パナソニック製ドラム式省スペース洗濯機の最適な購入先を決定
- 期限: 11月15日の引越しまでに購入・設置完了が必要
