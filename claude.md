## 📁 ファイル構成

### 1. `タスク一覧.md`
**目的**: 実行タスク一覧（チェックリスト形式）

引越しに必要なすべてのタスクを引っ越し前/後で分けて管理するファイルです。

**カテゴリー**:
- 🚨 最優先タスク（期限あり）
- 📋 引っ越し前タスク（契約・手続き、購入品、処分・整理など）
- 🏡 引っ越し後タスク（行政手続き、新居セットアップ、周辺情報確認）
- 📝 決定事項
- 📊 進捗状況

**使い方**: タスク完了時に `- [ ]` を `- [x]` に変更して進捗を管理

### 1-2. `未完了タスク一覧.md`（自動生成）
**目的**: 未完了タスクのみを抽出したファイル

`タスク一覧.md` から未完了タスク（`- [ ]`）のみを自動抽出したファイルです。
このファイルは `generate_uncompleted_tasks.sh` スクリプトによって自動生成されるため、**直接編集しないでください**。

**更新方法**:
1. `タスク一覧.md` でタスクを完了マーク（`- [x]`）
2. `./generate_uncompleted_tasks.sh` を実行
3. 未完了タスク一覧が自動更新される

**表示内容**:
- 未完了タスクのみ（完了済みタスクは除外）
- 進捗統計（総タスク数、完了数、未完了数、進捗率）
- 生成日時

### 1-3. `generate_uncompleted_tasks.sh`
**目的**: 未完了タスクを自動抽出するシェルスクリプト

タスク一覧.mdから未完了タスクだけを抽出して `未完了タスク一覧.md` を生成します。

**実行方法**:
```bash
cd "/Users/yamamotohideki/プロジェクト管理/引越し計画"
./generate_uncompleted_tasks.sh
```

**詳細**: `README-タスク管理.md` を参照

### 1-4. `タスク管理ルール.md`
**目的**: タスクのチェックルールを明文化

タスク管理における重要なルール：
- **子タスクがある親タスクは、すべての子タスクが完了するまで完了にしない**
- 親タスクと子タスクの整合性を保つ
- 進捗率の正確性を担保

### 2. `検討事項.md`
**目的**: 検討事項・ヒアリング項目

引越しに向けて考えるべき詳細事項をまとめたチェックリストです。各項目について検討し、決定事項を記入していきます。

**カテゴリー**:
- 🏠 物件・契約関係
- 📦 荷物・家電関連
- 🧾 行政・住所変更関連
- 🔌 インフラ・ライフライン関連
- 🪑 新居準備・初期設定
- 🧘 時間・体力・メンタル面
- 💬 今後の整理用（希望・制約）

### 3. `購入品メモ.md`
**目的**: 購入品のステータス管理

引越しに伴う購入品の現在のステータスをメモ書き形式で管理するファイルです。

**主な購入品**:
- ウォシュレット（購入先と工事業者を選定中）
- 洗濯機（パナソニック170,000円台で検討中、Amazon購入予定、パンサイズ測定待ち）
- カーテン（引っ越し後に検討）
- 自転車（駐輪場台数確認後に検討）

### 4. `アート引越センターへのヒアリングリスト.md`
**目的**: 見積もり訪問時の質問事項リスト

アート引越センターの見積もり訪問時に確認すべき質問事項をまとめたチェックリストです。

**主なカテゴリー**:
- プラン・料金関連（フルコースプランとハーフプランの比較）
- 洗濯機関連（処分費用、設置サービス）
- 引越し当日までの準備作業（解体、剥がす、整理の塩梅）
- スケジュール・作業内容
- 物件関連の確認（搬出入経路、駐車場）
- 特殊な荷物（ピアノ運搬）
- 料金・支払い関連
- 保険・補償

### 5. `議事録/`フォルダ
**目的**: ミーティング・説明会の議事録保管

引越しに関するミーティングや説明会の議事録を保管するフォルダです。

**保存済み議事録**:
- `2025-10-17-meeting.md` - 引越し準備タスクの棚卸しと段取り確認（初期対応、行政手続き、購入・処分計画など）
- `ハウスコムから契約まわり説明受.md` - 重要事項説明会の議事録と約42分間の完全な文字起こし

### 6. `資料/`フォルダ
**目的**: PDF等の資料保管

契約書、重要事項説明書、物件資料、見積書などのPDFファイルを保管するフォルダです。

### 7. `プロンプト/`フォルダ
**目的**: 他のAIに依頼するプロンプト管理

他のAI（o3 MCP、ChatGPTなど）にリサーチや調査を依頼する際のプロンプトをまとめて保管するフォルダです。

**保存済みプロンプト**:
- `洗濯機購入先リサーチ.md` - パナソニック製ドラム式省スペース洗濯機の購入先調査用
